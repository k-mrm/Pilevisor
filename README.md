# Pilevisor
aarch64 thin hypervisor for resource aggregation

## How to use

### requirement
- qemu-system-aarch64(>=6.2.0)
- binutils-aarch64-linux-gnu(>=2.38)

### build & run
- require sudo

```
# on terminal 1
$ make dev-sub
# on terminal 2
$ make dev-main
```
